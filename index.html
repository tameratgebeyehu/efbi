Certification Pathway

To qualify for an official EFBI Certificate of Completion, students must demonstrate practical capability by completing all criteria in their respective courses:

  • Lessons & Attendance

    Attend or view all interactive video lectures and participate in core Q&A.

  • Assignments & Labs

    Submit weekly practical programming labs and receive approvals from mentors.

  • Final Capstone Assessment

    Build, document, and present a functional technological solution solving a local problem.

Google Apps Script Deployment Guide

1
Open Script Editor in your Google Sheet

In your Google Sheet, go to Extensions > Apps Script. Replace all existing code with the contents of google_apps_script.js from this project.

2
Paste your Spreadsheet ID

Copy the long ID from the spreadsheet URL (between /d/ and /edit), then paste it into the SPREADSHEET_ID constant at the top of the script.

3
Deploy as Web App

Click Deploy > New Deployment. Set type to Web App. Execute as: Me. Access: Anyone. Click Deploy and authorize all permissions.

4
Paste the Web App URL above

Copy the Web App URL, paste it into the field above, set Mode to Live — Google Sheets Sync, and click Save Connection Settings. Your platform is now live! 🎉
